LibRedirect

A web extension that redirects YouTube, Instagram, Reddit, TikTok and other websites to alternative privacy-friendly frontends.

  • YouTube →  Invidious, Materialious, Piped, Piped-Material, Poke, CloudTube, LightTube, Tubo, FreeTube, Yattee, FreeTube PWA ViewTube
  • YouTube Music →  Beatbump, Hyperpipe
  • Twitter →  Nitter
  • Bluesky →  skyview
  • Reddit → Libreddit, Redlib, Teddit
  • Tumblr →  Priviblur
  • Twitch →  SafeTwitch, Twineo
  • TikTok →  ProxiTok
  • Instagram →  Proxigram
  • IMDb →  LibreMDb
  • Bilibili →  MikuInvidious
  • Pixiv →  PixivFE
  • Fandom →  BreezeWiki
  • Imgur →  Rimgo
