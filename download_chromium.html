Install for Chromium browsers (Chrome, Brave, Edge, Vivaldi, Opera)

Windows

1. Go to Extension Manager, and enable developer mode

2. Run this registery file: chrome.reg or brave.reg or edge.reg or vivaldi.reg.
Note: Opera doesn't need a registery file file.

3. Restart your browser

4. Right click on this .crx file, and click Save Link As... to save it: libredirect-2.8.1.crx

5. Go to Extension Manager again, and drag and drop the .crx file

MacOS

1. Download the latest zip file libredirect-2.8.1.zip

2. Unzip it with Auto detect subfolder

3. Go to Extension Manager, and enable developer mode

4. Click Load unpacked. Select and open libredirect-2.8.1/src/

Linux, ChromeOS

1. Right click on this .crx file, and click Save Link As... to save it: libredirect-2.8.1.crx

2. Go to Extension Manager, and enable developer mode

3. Refresh the page

4. Drag and drop the .crx file
